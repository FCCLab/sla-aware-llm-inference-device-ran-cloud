# SLA-Aware Distributed LLM Inference Across Device-RAN-Cloud
Scripts and derived logs for reproducing the latency/feasibility results in “SLA-Aware Distributed LLM Inference Across Device–RAN-Edge–Cloud” on a 5G SA AI-RAN testbed (MIG-isolated co-location).

Data license: The contents of the data/ directory (including derived per-request latency logs and sanitized metrics) are licensed under Creative Commons Attribution 4.0 International (CC BY 4.0).
Code license: All code, scripts, and documentation outside data/ are licensed under the MIT License unless otherwise noted.
